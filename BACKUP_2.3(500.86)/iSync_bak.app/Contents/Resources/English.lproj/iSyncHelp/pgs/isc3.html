I can't add a device to iSync
 

If you can't add a device to iSync, do the following:

•

Make sure your device is iSync-compatible. For a list of compatible devices, visit www.apple.com/isync/devices.html.

•

Make sure you've followed the instructions in iSync Help for setting up your device to work with iSync, and that it's currently connected or within Bluetooth range. For more information, click "Tell me more."

•

If the device is a Bluetooth or USB mobile phone, make sure that you're using the correct connection method for the device (for example, some Bluetooth phones can only sync with iSync using a USB cable). For more information, visit the iSync website at www.apple.com/isync/devices.html.

•

If the device is a Bluetooth mobile phone, make sure the phone is paired with your computer and that you can see it in the Paired Devices pane of Bluetooth preferences. For more information, search Mac Help (choose Library > Mac Help).

•

If the device is a USB mobile phone, make sure you can see it in the USB section of the About This Mac window. (Choose Apple menu > About This Mac, click the More Info button, and under Hardware select USB.) If your phone is connected to your computer and you don't see your phone listed in this window, there may be something wrong with your phone's USB cable or the connection to your computer.

•

If you're trying to add a Palm OS device, make sure you've enabled Palm syncing by choosing Devices > Enable Palm OS Syncing. For instructions on setting up your Palm OS device, click "Tell me more."

•

Quit iSync, then open it and try to add the device again.

•

If you have more than one Bluetooth device and you've connected a USB Bluetooth adapter to your computer, the adapter may not work with multiple Bluetooth connections. See the information that came with your adapter, or try using a different Bluetooth adapter.

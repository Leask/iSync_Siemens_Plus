Keyboard shortcuts
 

You can use your keyboard to quickly accomplish many tasks in iSync. To find the shortcuts for common commands, look in the menus (or see the menu shortcuts below). To complete an action, press the shortcut keys indicated below.

  Action Shortcut
iSync menu shortcuts
Open iSync Preferences Command-comma
Hide the iSync window Command-H
Hide other application windows Option-Command-H
Quit the iSync application Command-Q
Device menu shortcuts
Add Device Command-N
Synchronize now Command-T
Edit menu shortcuts
Undo your last typing action Command-Z
Redo your last typing action Shift-Command-Z
Cut the selected text Command-X
Copy the selected text Command-C
Paste the cut or copied text Command-V
Select all text Command-A
Open the Character Palette Option-Command-T
Window menu shortcuts
Put the iSync window in the Dock Command-M
Close the iSync window Command-W
Show the iSync window Command-1
Show the synchronization log Command-2
Help menu shortcuts
Open iSync Help Command-?
